# Backend for GEONIXA-EXAM

This folder contains a minimal Node + Express + TypeScript backend.

Quick start (from repository root):

```bash
cd backend
npm install
npm run dev
```

Build & run:

```bash
cd backend
npm run build
npm start
```

Endpoints:
- `GET /health` - health check
- `GET /api/ping` - ping endpoint

Add backend environment variables to Vercel/your host using `.env.example` as a reference.
