Prevention\n\n### How It Works\n\n1. **Randomized Testcases**: Each student gets unique random inputs\n2. **Deterministic**: Same student gets same inputs across retakes  \n3. **Detection**: If solution passes normal but fails randomized → **HARDCODING DETECTED**\n4. **Audit Trail**: All flags recorded with metadata\n\n```typescript\nconst randomized = RandomizedTestcaseGenerator.generateRandomizedArrayTestcases(\n  studentId,    // e.g., \"student@email.com\"\n  examId,       // e.g., \"exam-2024-01\"\n  questionId    // e.g., \"question-42\"\n);\n\n// Different students get different random values\n// Same student gets same values when retaking\n```\n\n## 📊 Admin Dashboard Metrics\n\nThe admin dashboard displays:\n\n```\n✅
